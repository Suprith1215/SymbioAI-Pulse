# PulseIQ - Cognizant Hackathon Project

## Overview
PulseIQ is a comprehensive system built for the Cognizant Hackathon, featuring an AI-driven backend and a modern React frontend.

## Project Structure
- **/backend**: Python-based AI pipeline and server.
- **/frontend**: React + Vite based user dashboard and UI components.

## Setup Instructions

### Backend
1. Navigate to `backend/`.
2. Install dependencies: `pip install -r requirements.txt`.
3. Run the server: `python main.py`.

### Frontend
1. Navigate to `frontend/`.
2. Install dependencies: `npm install`.
3. Start the development server: `npm run dev`.

## Features
- AI Decision Support Pipeline.
- Realistic 3D UI components.
- Integrated Simulation and Rule Editor.
